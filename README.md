# discordjs-nukebot

edit the #39 line with your bot token >> https://discord.com/developers/applications/
go to your application > bot and select all Privileged Gateway Intents options
install discord.js module with npm i discord.js command
if you not have node.js you must install this https://nodejs.org/en/
when bot will be online type n!help on chat to see all commands!
